# 영화 카드 게임 - 디자인 가이드

## 🎨 색상 시스템

### 배경 그라데이션
```css
bg-gradient-to-br from-slate-900 via-purple-900 to-slate-900
```
- 어두운 슬레이트에서 보라색으로 대각선 그라데이션
- 고급스럽고 몰입감 있는 배경

### 장르별 카드 색상
| 장르 | 색상 클래스 | 색상 |
|------|------------|------|
| 범죄 | `bg-red-600` | 빨간색 |
| 복수극 | `bg-purple-600` | 보라색 |
| 액션 | `bg-orange-600` | 주황색 |
| 스릴러 | `bg-gray-700` | 회색 |
| 사극 | `bg-amber-700` | 황갈색 |
| 멜로 | `bg-pink-600` | 분홍색 |
| 재난 | `bg-blue-700` | 파란색 |

### UI 요소 색상
- **수집 완료 버튼**: `bg-green-500` (초록색)
- **퀴즈 버튼**: `from-green-500 to-emerald-600` (초록 그라데이션)
- **정답**: `bg-green-100 border-green-500` (연한 초록)
- **오답**: `bg-red-100 border-red-500` (연한 빨강)

## 🎴 카드 디자인

### 카드 앞면
```
┌─────────────────────┐
│  [No.1]             │  ← 카드 번호 (우측 상단)
│                     │
│     [영화 아이콘]     │  ← 아이콘 (중앙)
│                     │
│    영화 제목         │  ← 큰 제목
│    [장르 배지]       │  ← 장르 표시
│                     │
│  "클릭하여 뒤집기"   │  ← 안내 문구 (하단)
└─────────────────────┘
```

**스타일 특징:**
- 반투명 효과: `bg-white/20 backdrop-blur-sm`
- 둥근 모서리: `rounded-2xl`, `rounded-full`
- 그림자: `shadow-2xl`

### 카드 뒷면
```
┌─────────────────────┐
│ 영화제목    [장르]   │  ← 헤더
├─────────────────────┤
│ 👤 감독              │  ← 감독 정보
│   감독 이름          │
├─────────────────────┤
│ 👥 출연진            │  ← 출연진 정보
│   배우 이름들...     │
├─────────────────────┤
│ [카드 수집하기]      │  ← 버튼
└─────────────────────┘
```

**정보 섹션 디자인:**
- 감독: `from-blue-50 to-blue-100` (파란 그라데이션)
- 출연진: `from-purple-50 to-purple-100` (보라 그라데이션)
- 아이콘과 함께 표시

## 🔄 애니메이션 효과

### 카드 뒤집기
```css
transition-transform duration-700
transform: rotateY(180deg)
transform-style: preserve-3d
backface-visibility: hidden
```
- 3D 회전 효과 (Y축 기준)
- 0.7초 부드러운 전환
- 뒷면 숨김 처리

### 호버 효과
- 컬렉션 카드: `hover:bg-white/30`
- 버튼: `hover:from-purple-700 hover:to-pink-700`

### 진행 바 애니메이션
```css
transition-all duration-300
```
- 퀴즈 진행률 부드러운 전환

## 🧠 퀴즈 화면 디자인

### 결과 점수 배지
```
     ┌────────┐
     │  85%   │  ← 큰 점수 표시
     └────────┘
```
- 80% 이상: 초록색 (`bg-green-500`)
- 60-79%: 노란색 (`bg-yellow-500`)
- 60% 미만: 빨간색 (`bg-red-500`)

### 문제 카드
```
┌─────────────────────────┐
│ [나가기]    문제 1/5     │  ← 헤더
├─────────────────────────┤
│ ▓▓▓▓▓░░░░░░░░░░░      │  ← 진행 바
├─────────────────────────┤
│                         │
│   질문이 여기 표시됩니다  │
│                         │
├─────────────────────────┤
│ [ 선택지 1 ]            │
│ [ 선택지 2 ]            │
│ [ 선택지 3 ]            │
│ [ 선택지 4 ]            │
├─────────────────────────┤
│   현재 점수: 3점         │
└─────────────────────────┘
```

### 정답/오답 표시
- 정답 선택: 초록 배경 + ✓ 체크 아이콘
- 오답 선택: 빨강 배경 + ✗ X 아이콘
- 정답 강조: 항상 초록색으로 표시

## 📱 반응형 디자인

### 브레이크포인트
- 기본 (모바일): `< 768px`
- `md` (태블릿): `≥ 768px`
- `lg` (데스크톱): `≥ 1024px`

### 텍스트 크기 조정
- 메인 제목: `text-5xl` (3rem)
- 카드 제목: `text-3xl` → `text-2xl` (카드 뒷면)
- 본문: `text-lg` → `text-sm`

### 패딩 조정
- 전체 페이지: `p-8`
- 카드 내부: `p-6`
- 정보 섹션: `p-4`

## 🎯 타이포그래피

### 폰트 굵기
- 제목: `font-bold` (700)
- 강조: `font-semibold` (600)
- 본문: 기본 (400)

### 텍스트 색상
- 밝은 배경: `text-gray-800` (제목), `text-gray-600` (본문)
- 어두운 배경: `text-white`, `text-purple-200`

## ✨ 특수 효과

### 유리 효과 (Glassmorphism)
```css
bg-white/10 backdrop-blur-sm
bg-white/20 backdrop-blur-md
```
- 반투명 배경 + 블러 효과
- 현대적이고 세련된 느낌

### 그림자
- 카드: `shadow-2xl` (큰 그림자)
- 버튼: `shadow-lg` (중간 그림자)

### 둥근 모서리
- 카드: `rounded-2xl` (1rem)
- 배지/버튼: `rounded-full` (완전한 원형)
- 섹션: `rounded-lg` (0.5rem)

---

## 🎨 디자인 원칙

1. **일관성**: 모든 카드와 버튼이 동일한 스타일 패턴 사용
2. **계층 구조**: 크기와 색상으로 정보의 중요도 구분
3. **피드백**: 모든 인터랙션에 시각적 피드백 제공
4. **가독성**: 충분한 대비와 여백으로 읽기 쉽게
5. **몰입감**: 그라데이션과 애니메이션으로 게임 분위기 조성
