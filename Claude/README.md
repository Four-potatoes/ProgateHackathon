# Claude 폴더 - 프로젝트 관리 문서

이 폴더는 Progate Hackathon 프로젝트의 전체 관리 문서를 담고 있습니다.

## 📂 디렉토리 구조

```
Claude/
├── README.md                    # 본 파일
├── docs/
│   ├── project-overview.md      # 프로젝트 개요
│   ├── implementation.md        # 구현 상세 문서
│   └── team-contributions.md    # 팀원 기여도 (예정)
└── assets/
    └── screenshots/             # 스크린샷 및 데모 이미지 (예정)
```

## 📄 문서 목록

### 1. project-overview.md
**프로젝트 전체 개요**
- 프로젝트 정보 (주제, 이름)
- 핵심 기능 설명
- 수록 영화 목록
- 기술 스택
- 팀 기여도
- 평가 기준 대응
- 향후 개선 사항

### 2. implementation.md
**기술 구현 상세**
- 아키텍처 다이어그램
- 데이터 구조
- 핵심 게임 로직
- UI/UX 구현
- AI 학습 시스템
- Firebase 연동
- 페이지 라우팅
- 최적화 전략
- 테스트 시나리오

### 3. team-contributions.md (예정)
**팀원별 세부 기여도**
- 각 팀원의 작업 내용
- 코드 기여도
- 시간별 작업 로그

## 🎯 문서 활용 방법

### 발표 준비
1. `project-overview.md`에서 전체 흐름 파악
2. `implementation.md`에서 기술 세부사항 확인
3. 데모 시나리오 준비

### 코드 리뷰
1. `implementation.md`의 아키텍처 섹션 참고
2. 각 기능별 구현 방식 이해
3. 최적화 전략 확인

### 향후 개발
1. "향후 개선 사항" 섹션 확인
2. Phase별 우선순위 파악
3. 기술 스택 확장 계획 수립
