# K-Movie Memory Master - 프로젝트 개요

## 📋 프로젝트 정보

### 주제
**K-문화와 기술을 융합시켜서 서비스 제작**

### 프로젝트명
**K-Movie Memory Master** - 한국 영화를 테마로 한 카드 매칭 게임

---

## 🎯 핵심 기능

### 1. 카드 매칭 게임 (메인 기능)
- 한국 명작 영화 8편을 주제로 한 메모리 카드 게임
- 4x4 그리드 (총 16장)
- 카드 뒤집기 애니메이션 효과
- 시도 횟수 및 남은 쌍 추적

### 2. 영화 도감 (컬렉션)
- 수록된 모든 한국 영화 정보 열람
- 영화 포스터 이미지
- 장르, 감독, 출연진 정보
- 영화 설명

### 3. AI 기반 학습
- 영화 정보를 기반으로 한 퀴즈 생성
- 사용자 답변에 대한 즉각적인 피드백
- 정답/오답 시 상세 정보 제공
- LLM 연동 준비 완료

### 4. 랭킹 시스템
- Firebase Firestore 기반 실시간 리더보드
- 최소 시도 횟수 기준 Top 10
- 익명 사용자 식별 시스템

## 🛠 기술 스택

### Frontend
- **HTML5** - 시맨틱 마크업
- **Tailwind CSS** - 유틸리티 기반 스타일링

- **CSS 3D Transforms** - 카드 뒤집기 효과

## 🎮 사용 방법

1. **게임 시작**
   - index.html을 브라우저에서 실행
   - 카드를 클릭하여 뒤집기
   - 같은 영화 쌍을 찾아 매칭

2. **영화 도감 확인**
   - 하단의 "영화 도감" 버튼 클릭
   - 모든 영화 정보 열람

3. **AI 학습**
   - "AI 학습" 버튼 클릭
   - 영화 설명을 보고 제목 맞추기
   - 정답 확인 후 다음 문제로 이동

4. **랭킹 확인**
   - 게임 완료 후 자동으로 점수 제출
   - "랭킹" 버튼으로 리더보드 확인

## 🚀 향후 개선 사항

1. **LLM 연동**: OpenAI/Anthropic API를 통한 실시간 AI 퀴즈 생성
2. **난이도 조절**: Easy(3x2), Normal(4x4), Hard(6x6)
3. **추가 컨텐츠**: K-Pop, K-Drama, K-Food 테마
4. **소셜 기능**: 친구와 점수 비교, 공유 기능
5. **다국어 지원**: 영어, 일본어, 중국어 번역
